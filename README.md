## 🚀 How to Deploy on GitHub Pages

### Step 1 — Create Repository
```
1. Go to github.com → click "+" → New Repository
2. Name it: devops-roadmap
3. Set to Public
4. Click "Create repository"
```

### Step 2 — Upload Files
```
1. Click "Add file" → "Upload files"
2. Upload ALL files keeping the folder structure intact:
   - index.html (root)
   - css/style.css
   - js/main.js
   - js/progress.js
   - pages/roadmap.html
   - pages/projects.html
   - pages/stack.html
   - pages/progress.html
3. Click "Commit changes"
```

### Step 3 — Enable GitHub Pages
```
1. Go to Repository Settings
2. Scroll to "Pages" section
3. Source → select "main" branch → root folder
4. Click Save
5. Wait ~2 minutes → your site is LIVE! 🎉
```

### Step 4 — Using Git CLI (Optional)
```bash
git clone https://github.com/yourusername/devops-roadmap.git
cd devops-roadmap
# Copy all your files here
git add .
git commit -m "🚀 Launch DevOps AI Roadmap"
git push origin main
```
